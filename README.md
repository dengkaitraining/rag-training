### 「Python Flask 框架」與「檢索增強生成應用系統開發」課程範例
#### 開發與測試環境部署
 - 1. 建立 python 虛擬環境。
```sh
# 不要使用 sudo 管理者權限安裝
# 建立 python 虛擬環境 venv
python -m venv venv
```
 - 2. 安裝必要套件

| python 模組 | 說明 |
| :---| :--- |
|     |      |

```sh
# 不要使用 sudo 管理者權限安裝
python -m pip install numpy
python -m pip install flask
python -m pip install requests
python -m pip install markdown
python -m pip install dotenv
```
