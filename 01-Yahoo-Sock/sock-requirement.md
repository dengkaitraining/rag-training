# 如何建立台灣公司相關資訊系統，用於股票投資產業方面資訊：
## 使用者需求：
### 1. 資訊取得
 - 透過網際網路、youtube取得公司產業資訊、生產產品商品資訊、ESG永續經營要指標、碳權等資訊。
 - 透過網際網路、youtube取得公司股票技術面資訊。
 - 透過網際網路、youtube取得公司產業新聞資訊。
 - 前項取得的資料可儲存與本地端資料庫，並每天從網際網路、youtube更新全部最新資訊。
### 2. 使用者節色
 - 一般使用者 (User)：系統主要使用者，僅提供查詢{1. 資訊取得}的資料內容。
 - 系統管理員 (Admin)：系統後台管理者，新增/編輯會{1. 資訊取得}的資料內容、查詢使用率統計。
## 系統技術堆疊：
### 1. 系統部署環境
 - 使用 docker 容器化技術建構環境：網頁伺服器使用 apache httpd、資料庫使用 PostgreSQL、資料庫連線 Redis、程式開發使用 python diango。
### 2. 網頁前端技術
 - 程式語言：TypeScript
 - 核心框架：Vue.js
 - 樣式與 UI：Tailwind CSS
### 3. 網頁後端技術
 - 程式語言：Python
 - 核心框架：Django
